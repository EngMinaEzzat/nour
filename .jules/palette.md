## 2023-10-27 - Palette: Cloud Glow Theme
**Learning:** Adding new themes to the application requires updating types, defining presets, and carefully mapping fallback default sections without altering existing defaults. The conditional mappings in the section factory must correctly cascade styles based on the selected theme, and all localization files must reflect the updated copy keys.
**Action:** Always add `style` arguments to section factory functions and conditionally apply keys. Provide translations for all dynamic components (hero, about, lookbook, trust strips) across all active locales.
