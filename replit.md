# نور — Egyptian Fashion & Beauty SaaS Platform

## Overview

نور (Nour) is a multi-tenant SaaS platform designed for Egyptian fashion and cosmetics merchants. It enables them to create and manage their own online stores, similar to Shopify, but with a focus on the Egyptian market. The platform offers a full Arabic RTL UI, advanced storefront customization, and comprehensive administrative tools.

**Key Capabilities:**
- **Storefront Management:** Merchants can create and customize their online stores using a visual WYSIWYG editor and pre-designed section components.
- **Product & Order Management:** Full CRUD operations for products, categories, and robust order management with atomic transactions, stock decrement, and status history.
- **Customer & Staff Management:** Features a customer directory, staff management with various roles, and invitation systems.
- **Analytics & Reporting:** Provides detailed analytics, platform health scores, and data export functionalities.
- **SaaS Billing:** Integrates with Paymob for payments, handles subscription management, invoicing, and bank transfer requests.
- **Marketing & SEO:** Includes features for tracking pixels, custom domain management, and SEO controls with live SERP previews.
- **Communication:** Integrates with WhatsApp for order updates, follow-ups, and customer communication.
- **Automation:** Allows for the creation of trigger-action rules for various store operations.

The platform aims to empower Egyptian entrepreneurs in the fashion and beauty sectors by providing a localized, feature-rich, and scalable e-commerce solution.

## System Architecture

The system is built as a monorepo using `pnpm workspaces`, Node.js 24, and TypeScript 5.9.

**Frontend (fashion-store):**
- **Technology:** React, Vite, Framer Motion for animations.
- **UI/UX:** Full Arabic RTL interface, uses Cairo, Tajawal, and Cormorant Garamond fonts.
- **Design:** Premium storefront redesign with a blush/ivory/champagne palette and serif headings. Includes 15 customizable section components.
- **Store Builder:** A 6-step onboarding wizard and a full visual WYSIWYG editor for store customization, featuring an AI assistant panel, readiness checklist, and 30-item undo/redo history.
- **Page Meta:** Utilizes `usePageMeta` hook for dynamic SEO, Open Graph tags, Twitter cards, canonical links, and JSON-LD structured data on storefront pages.

**Backend (api-server):**
- **Framework:** Express 5.
- **Database:** PostgreSQL with Drizzle ORM.
- **Validation:** Zod, with schemas generated by Orval.
- **API Codegen:** Orval generates React Query hooks and Zod schemas from an OpenAPI spec.
- **Authentication:** `express-session` with `connect-pg-simple` for session management and `bcryptjs` for password hashing.
- **Authorization:** Role-based access control (`requireRole`, `requirePlatformAdmin`) for various merchant and platform admin roles.
- **Tenant Isolation:** Robust tenant isolation enforced across all API routes to prevent cross-tenant data access.
- **Image Uploads:** Secure image upload system with magic-byte validation, UUID renaming, and static serving.
- **AI Guardrails:** Implemented for AI features, including system prompt locking, input truncation, token limits, and per-merchant rate limiting.
- **Security:** CSRF protection using a double-submit cookie pattern.

**Core Features & Implementations:**
- **Entitlements:** Plan definitions (`starter`, `growth`, `pro`) are centrally managed, dictating product limits, order volumes, staff count, and WhatsApp access.
- **SaaS Billing:** Comprehensive system for subscription management, invoicing, and handling bank transfer requests, integrated with Paymob.
- **Custom Domains:** Functionality for merchants to add, verify, and manage custom domains for their stores.
- **Tracking Pixels:** Integration for popular marketing pixels (Meta, Google Analytics, TikTok, Snapchat) with a dynamic snippet generation for storefront injection.
- **Data Exports:** Asynchronous job system for exporting orders, products, and customer data in CSV format.
- **Platform Administration:** Dedicated console for platform operators to manage merchants, monitor health scores, and manage provider health.
- **Performance & Scalability:** Implemented Redis for session storage, in-memory subscription cache, database indexing, and various rate limiters (storefront, checkout, export, AI).
- **SEO/SSR:** Bot prerender endpoint detects crawlers and serves minimal SSR HTML with rich OG meta and JSON-LD for improved SEO.
- **Egyptian Localization:** Utilities for Egyptian phone number validation and normalization, EGP currency formatting, and a comprehensive list of Egyptian governorates. All user-facing error messages are in Arabic.

- **Payment Gateway:** Paymob

- **Carrier Integration:** Bosta (for shipping)
- **WhatsApp API:** Integrated for messaging (provider credentials managed server-side)
